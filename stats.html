Info

Overview

Once the extension is opened by clicking on the icon, it locates each Active Variant (up to 10) on the Unbounce page and then pulls in each Active Variant's Visitors and Views data as well as the data for the Champ Starting Visitors and Champ Starting Conversions if you have that data listed accurately in each Active Variant's Title, and then it calculates everything instantly for you without any input or action from the user so you can see the true stats, winners, and losers without needing a calculator or spreadsheet. This saves a lot of time if you do a lot of testing, and enables you to do more testing now that calculations are automatically done for you.

Results

Once each Active Variant has been calculated, the Result will populate with "Analyzing", "Winning", or "Losing."

  • If an Active Variant has not reached 30 Visitors, the Result will display "Analyzing" which means you need a few more views before we recommend making a decision on the success of that Active Variant.
  • If an Active Variant has reached 30 Visitors and the Variant Conversion Rate is greater than the Champ Conversion Rate, the Result will display "Winning" which means your split test is currently successful.
  • If an Active Variant has reached 30 Visitors and the Variant Conversion Rate is less than the Champ Conversion Rate, the Result will display "Losing" which means your split test is currently not successful.

Why the 30 Visitors benchmark? This benchmark comes from the MintCRO Methodology in regards to working with small data so we can gain faster "wins" in our quest for faster and more profitable conversion rate optimization in our marketing. If you would like to change the 30 Visitors benchmark, you may do so in settings.

Champion Starting Data

If you have a champion variant that has been running for a while that has received quite a bit of traffic and you are still testing more variants trying to find a new champion, then the stats for your champion variant aren't truly accurate when you compare it against your new variants because your champion variant stats are including traffic that was sent to it during the time many other previous variants were active, so it isn't fair to compare it against the current variant because the current variant has much less traffic to analyze and compare thus making the conversion rate uncomparable.

This is the main reason this extension was created, because the Champion stats shown in Unbounce aren't exactly the stats needed to make a more informed decision concerning which variant is perfoming better, the Champion or the new test variant. In order to allow the extension to calculate the Champion variant stats from only after the new test variant was launched, you need to include the Champion Starting Visitors and Champion Starting Conversions of when the new test variant is launched in the actualy title of the new test variant.

For example:

Variant B - Title Goes Here: Added large review widget | (1339.csV 52.csC)
Variant C - Title Goes Here: Added small review widget | (1221.csV 43.csC)

Doesn't have to be in parentheses, but I like to do that. Remove commas from numbers and make sure the Champion Starting Visitors number is followed by a ".csV" and make sure the Champion Starting Conversions number is followed by a ".csC" for the calculations to be accurate.
