Overview

Once the extension is opened, it locates each Active Variant (up to 10) on the Unbounce page and then pulls in each Active Variant's Visitors and Views data as well as the data for the Champ Starting Visitors and Champ Starting Conversions if you have that data listed accurately in each Active Variant's Title, and then it calculates everything instantly for you without any input or action from the user so you can see the true stats, winners, and losers without needing a calculator or spreadsheet.

Results

Once each Active Variant has been calculated, the Result will populate with "Analyzing", "Winning", or "Losing."

  • If an Active Variant has not reached 30 Views, the Result will display "Analyzing" which means you need a few more views before we recommend making a decision on the success of that Active Variant.
  • If an Active Variant has reached 30 Views and the Variant Conversion Rate is greater than the Champ Conversion Rate, the Result will display "Winning" which means your split test is currently successful.
  • If an Active Variant has reached 30 Views and the Variant Conversion Rate is less than the Champ Conversion Rate, the Result will display "Losing" which means your split test is currently not successful.

Why the 30 Views benchmark? This benchmark comes from the MintCRO Methodology in regards to working with small data so we can gain faster "wins" in our quest for faster and more profitable conversion rate optimization in our marketing.
